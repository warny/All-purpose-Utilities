Replace<T>(T[], T[], T[])

Replaces non-overlapping occurrences of the subsequence toReplace in array with the subsequence replacement.

public static T[] Replace<T>(this T[] array, T[] toReplace, T[] replacement) where T : IEquatable<T>

Parameters

array T[]

The source array in which to perform replacements.

toReplace T[]

The subsequence to find and replace.

replacement T[]

The subsequence to substitute in place of toReplace.

Returns

T[]

A new array with the replacements applied where the pattern is matched.

Type Parameters

T

Type of elements, must implement IEquatable<T> for equality checks.

Remarks

Overlapping matches are not detected. Only distinct, non-overlapping occurrences are replaced.

If toReplace is empty, no replacement is done and the original array is returned as-is.

Exceptions

ArgumentNullException

Thrown if array, toReplace, or replacement is null.
