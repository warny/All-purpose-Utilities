Namespace Utils.Net.DNS.RFC1183

Classes

AFSDB

Represents an AFSDB (Andrew File System Database) record, as defined in RFC 1183 §1.2. This record helps locate AFS or related database servers for a particular domain.

ISDN

Represents an ISDN (Integrated Services Digital Network) DNS record, as originally defined in RFC 1183 §3.2. This record stores an ISDN telephone number (and optionally a subaddress).

RP

Represents an RP (Responsible Person) record, as defined in RFC 1183 §2.2. This record identifies the mailbox of the person responsible for a particular DNS zone and an optional TXT record with additional information.

RT

Represents an RT (Route Through) record, as defined in RFC 1183 §2.3. The RT record specifies an intermediate host that should be used when routing to the owner of the record, along with a preference value.

X25

Represents an X.25 PSDN address record, as defined in RFC 1183 §3.1 (type code 19). This record stores a single X.25 PSDN address (e.g., a numeric string like "311061700956").
