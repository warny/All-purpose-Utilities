Remarks

MG records store a DNSDomainName (MGName) for a mailbox that is a member of a mail group. Unlike some other mail-related records, MG does not trigger additional section processing in DNS resolution.

The record type is 0x08 for IN. Although it may still be encountered, MG is considered part of an older, less common suite of experimental mail group DNS records. Modern mail routing largely relies on MX, but MG is included here for completeness.

The RDATA layout is:

+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+
/                   MGMNAME                     /
/                                               /
+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+

MGMNAME is a <domain-name> for a mailbox in the group.
