Struct Tag

Namespace
Utils.Fonts.TTF
Assembly
Utils.Fonts.dll

Represents a 4-byte identifier used in font file formats such as TrueType (TTF). Encodes a string of up to 4 ASCII characters into a 32-bit integer. Provides implicit conversions and equality comparisons with strings, integers, and byte arrays.
