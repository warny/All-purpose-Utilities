Provides an abstract base class for a virtual processor that interprets and executes instructions from a byte array-based context. Instructions are defined by methods annotated with InstructionAttribute.

VirtualProcessor(bool)

protected VirtualProcessor(bool littleEndian = true)

Parameters

littleEndian bool

If true, uses little-endian byte ordering; otherwise, uses big-endian. Defaults to true.
