ReadBlock(Stream, byte[])

Reads and returns the next "block" from the stream, delimited by the specified byte sequence. The returned array does not include the delimiter itself. If the delimiter is not found before EOF, all remaining bytes in the stream are returned.

This method can be called repeatedly to read multiple blocks from the same stream.

public static byte[] ReadBlock(this Stream s, byte[] blockSeparator)

Parameters

s Stream

The Stream from which to read the block.

blockSeparator byte[]

A byte array representing the block delimiter.

Returns

byte[]

A byte array containing the next block of data, not including the delimiter. If EOF is reached before finding the delimiter, returns all remaining data (which could be empty).

Exceptions

ArgumentNullException

Thrown if blockSeparator is null.

ArgumentException

Thrown if blockSeparator is empty (cannot search for an empty delimiter).

ReadBytes(Stream, int, bool)

public static byte[] ReadBytes(this Stream s, int length, bool raiseException = false)

Parameters

s Stream

The source Stream to read from.

length int

The number of bytes to read from the stream.

raiseException bool

If true and fewer than length bytes could be read, throws an EndOfStreamException.

Returns

byte[]

A byte array containing up to length bytes read from the stream.
If raiseException is false and the stream ends, the returned array might be partially uninitialized for the remainder.

Exceptions

EndOfStreamException

Thrown if raiseException is true and the stream ends before length bytes could be read.
