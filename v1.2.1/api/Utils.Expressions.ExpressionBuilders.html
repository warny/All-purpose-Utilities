Namespace Utils.Expressions.ExpressionBuilders

Classes

AddAssignationBuilder

Implements IFollowUpExpressionBuilder for an additive assign operation (e.g. "+="), supporting string concatenation when the left operand is a string.

AssignationBuilder

Implements IFollowUpExpressionBuilder for simple assignment operators (e.g. "="), allowing a custom assignment BinaryExpression factory.

BlockBuilder

Implements IStartExpressionBuilder for a block expression, e.g. statements within { ... }. Each expression read inside becomes part of the block, using a BlockExpressionBuilder.

BracketBuilder

Implements IFollowUpExpressionBuilder for bracket-based operations, which may represent either array indexing or default member (indexer) access.

BreakBuilder

Implements IStartExpressionBuilder for the "break" statement, returning an Break(LabelTarget) to the nearest loop label.

CloseBuilder

Implements IFollowUpExpressionBuilder that checks for matching wrap start and sets the flag, concluding the bracket/parenthesis context.

ConditionalBuilder

Implements IFollowUpExpressionBuilder and IAdditionalTokens for a ternary-like operation, e.g., "condition ? expr : expr". The symbol recognized for the second separator is customizable via ChoiceSymbol.

ContinueBuilder

Implements IStartExpressionBuilder for the "continue" statement, returning an Break(LabelTarget) that effectively simulates a continue (jumping to the loop's continue label).

DefaultUnaryBuilder

A default unary builder that determines whether the token is a name (resolving a parameter, constant, or type) or a string/char literal, and creates the appropriate Expression.

FalseBuilder

Builds an expression representing false.

ForBuilder

Implements IStartExpressionBuilder for a "for" loop construct, building an Loop(Expression) with initializers, condition, and increments.

ForEachBuilder

Implements IStartExpressionBuilder for a "foreach" loop, building either an BlockExpression that enumerates an array or uses IEnumerator for other enumerables.

IfBuilder

Implements IStartExpressionBuilder for "if" statements, optionally handling an "else" branch.

MemberBuilder

Implements IFollowUpExpressionBuilder to handle a member access operation (e.g., appending ".MemberName" after an existing expression).

NewBuilder

Builds a new expression for either an object, array, or collection/member initialization.

NewBuilder.ReadNextExpressionBuilder

Builds an expression by reading the next expression from the parser at the given priority. Useful for sub-scenarios within "new" statements.

NewBuilder.UnaryMinusBuilder

Implements unary minus builder for negative numeric constants or unary expressions.

NullBuilder

Builds an expression representing null.

NullOrMemberBuilder

Implements IFollowUpExpressionBuilder for the null-conditional operator (?.), returning either null or the expression member access.

NumberConstantBuilder

Builds an expression representing a numeric constant. Handles base prefixes (e.g., "0x" for hex), numeric suffixes (e.g., "f" for float), and direct numeric literals, returning the corresponding ConstantExpression.

OperatorBuilder

Implements IFollowUpExpressionBuilder for binary operators, allowing a custom operator function and optional numeric type adjustment.

ParenthesisBuilder

Implements IStartExpressionBuilder for handling parentheses or bracket expressions, such as grouped sub-expressions, type casts, or lambda parameter lists.

PlusOperatorBuilder

Implements IFollowUpExpressionBuilder to handle the plus operator (+), supporting both numeric addition and string concatenation.

PostOperationBuilder

Implements IFollowUpExpressionBuilder for post-increment or post-decrement operators, e.g. i++ or i--. The supplied operator function is responsible for creating the appropriate unary expression.

ReturnBuilder

Implements IStartExpressionBuilder for the "return" statement. The builder simply reads the following expression and returns it, ignoring early-exit semantics.

RightParenthesisBuilder

Implements IFollowUpExpressionBuilder for a closing parenthesis builder. Depending on the context, this may represent function invocation or fallback logic.

SizeOfBuilder

Builds an expression for sizeof(T), retrieving the size of the specified type via SizeOf(Type).

SwitchBuilder

Implements IStartExpressionBuilder for switch statements. This builder supports both expression-style switches and the classic statement form where each case ends with break;.

ThrowParseException

Provides an implementation of both IStartExpressionBuilder and IFollowUpExpressionBuilder that throws a ParseUnknownException whenever an expression is to be built, indicating that the token or symbol cannot be parsed.

TrueBuilder

Builds an expression representing true.

TypeCastBuilder

Implements IFollowUpExpressionBuilder for type-cast operations such as (Type)expression. It uses Expression.TypeAs to create a safe cast expression.

TypeMatchBuilder

Implements IFollowUpExpressionBuilder for a type-check operator, e.g. "expression is Type".

TypeofBuilder

Builds an expression for typeof(T), returning a constant Type object.

UnaryOperandBuilder

Implements IStartExpressionBuilder for unary operators, building a unary expression using the provided OperatorBuilder function.

WhileBuilder

Implements IStartExpressionBuilder for a "while" loop construct, building an Loop(Expression). The loop terminates if the condition is false.
