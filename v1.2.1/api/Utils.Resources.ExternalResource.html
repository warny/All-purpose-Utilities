Class ExternalResource

Namespace
Utils.Resources
Assembly
Utils.dll

A cross-platform .resx reader that does not rely on System.Windows.Forms. It manually parses .resx XML to load both inline string data and external file references (similar to ResXFileRef).

This class implements IReadOnlyDictionary<TKey, TValue> so that it behaves like a read-only resource collection.
