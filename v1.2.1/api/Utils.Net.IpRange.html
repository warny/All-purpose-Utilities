Class IpRange

Namespace
Utils.Net
Assembly
Utils.dll

Represents a contiguous range of IP addresses. Instances are immutable and may be constructed from a pair of addresses or from a single address with a prefix length. The range can be enumerated to list each address it contains.

public class IpRange : IParsable<IpRange>, IEnumerable<IPAddress>, IEnumerable, IEquatable<IpRange>, IEqualityOperators<IpRange, IpRange, bool>
