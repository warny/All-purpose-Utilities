Namespace Utils.Fonts.PostScript

Classes

CidKeyedFont

Very small loader for CID-Keyed Type 1 PostScript fonts. Only a subset of the format is implemented. Refer to Adobe Technical Note 5014 for the full specification.

PostScriptFont

Very small PostScript font loader based on a custom text representation. The intention is to mimic a subset of the Type 1 syntax without parsing the full PostScript language. Each glyph is declared as shown below:

Glyph: A
Width: 500
Height: 600
Baseline: 0
Path:
M 0 0
L 10 0
...
EndGlyph

For a description of the real Type 1 format see Adobe Technical Note 5040.

PostScriptGlyph

Represents a glyph defined by a sequence of basic PostScript path commands. The commands correspond loosely to the operators described in Adobe's Type 1 specification.

Type3Font

Minimal loader for PostScript Type 3 fonts. Only a tiny subset of the language is interpreted: setcachedevice for metrics and the basic path commands moveto, lineto, curveto and closepath. See Adobe Technical Note 5015 for the full details.

Type42Font

Minimal loader for PostScript Type 42 fonts which encapsulate a TrueType font. This implementation extracts the embedded TrueType data from the /sfnts array and delegates glyph retrieval to TrueTypeFont. For the official format see Adobe Technical Note 5012.

PostScriptGlyph.PathCommand

Represents a single drawing command extracted from the font. The coordinates use the same units as the original charstring.
