Remarks

This class behaves similarly to UriBuilder but offers additional utility methods for:

  • Working with basic authorization (username/password).
  • Excluding credentials from the final URI when needed.
  • Auto-excluding well-known default ports from the URI.

Encodes and returns a Basic Authorization header value using Username and Password.

public string GetBasicAuthorizationBase64String()

Returns

string

A Base64-encoded string representing basic authorization credentials, or an empty string if no credentials are present.

Remarks

The format returned is a Base64-encoded string of "username:password". Returns an empty string if both Username and Password are null or whitespace.
