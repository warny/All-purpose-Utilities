Call(int)

Pushes a new frame onto the stack, recording returnAddress as the instruction pointer to restore when the corresponding Return() is called.

public void Call(int returnAddress)

Parameters

returnAddress int

Byte offset in the instruction stream to return to. Must be non-negative; negative values are reserved as termination sentinels and are therefore rejected.

Exceptions

ArgumentOutOfRangeException

Thrown when returnAddress is negative.

InvalidOperationException

Thrown when Depth has reached MaxDepth.

Return()

Pops the current frame and returns the saved return address, or -1 when the stack is empty. Because Call(int) rejects negative addresses, a return value of -1 unambiguously signals stack underflow/program termination; callers should assign it to InstructionPointer so the VirtualProcessor<T> execution loop stops.

public int Return()

Returns

int

The instruction-pointer value saved by the matching Call(int) (always ≥ 0), or -1 if the stack is empty.
