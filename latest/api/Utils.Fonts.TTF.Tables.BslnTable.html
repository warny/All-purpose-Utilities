The baseline table (tag: 'bsln') defines the positions of baselines for each script in the font, allowing mixed-script text to align glyphs from different writing systems on a common line.

[TTFTable(TableTypes.Tags.BSLN, new TableTypes.Tags[] { })]

Properties

ControlPoints

Gets or sets the 32-element control-point index array (one UInt16 per baseline slot). Each element is a control point index within StandardGlyph, or NoControlPoint (0xFFFF) if that baseline has no control point. Applies to ControlPointNoMap and ControlPointWithMap.

public ushort[] ControlPoints { get; set; }

Property Value

ushort[]

DefaultBaseline

Gets or sets the default baseline class (0–31) applied to all glyphs not covered by the per-glyph lookup (used by formats 1 and 3) or to all glyphs (formats 0 and 2).

public ushort DefaultBaseline { get; set; }

Property Value

ushort

Deltas

Gets or sets the 32-element delta array (one Int16 FUnit value per baseline slot). Each element is the distance from the natural baseline of glyphs assigned to that baseline class to the baseline position itself. Applies to DistanceNoMap and DistanceWithMap.

public short[] Deltas { get; set; }
