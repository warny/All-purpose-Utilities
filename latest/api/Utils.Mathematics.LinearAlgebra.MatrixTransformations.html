Remarks

Delegates to Diagonal(params IEnumerable<T>), which is the single implementation of diagonal-matrix construction: an earlier, independent implementation here cached isTriangular/isDiagonal as false whenever any diagonal value was zero, even though a matrix with zero diagonal entries is still diagonal and triangular by definition (only its invertibility/determinant changes). Two independently maintained factories for the same construction had already drifted into different, incorrect behavior; keeping one implementation avoids a repeat.

Rotation<T>(params IEnumerable<T>)

Generates a rotation matrix.

public static Matrix<T> Rotation<T>(params IEnumerable<T> angles) where T : struct, IFloatingPoint<T>, ITrigonometricFunctions<T>, IRootFunctions<T>

Parameters

angles IEnumerable<T>

Angles of rotation, one per axis pair of the base d × d rotation block: d * (d - 1) / 2 angles are required for a given base dimension d. Must not be empty: with zero angles, the d * (d - 1) / 2 = n formula solves to the degenerate d = 1 (a meaningless "1×1 rotation", since rotation requires at least two axes to rotate between) rather than identifying which ambient dimension's identity rotation the caller intended (see TODO-2026-07-11-pass5.md item #66). To build an identity matrix of a specific dimension, call Identity<T>(int) directly instead.

Returns

Matrix<T>

New rotation matrix.

Type Parameters

T

Numeric type of the matrix.

Exceptions

ArgumentException

Thrown when angles is empty, or its count does not match d * (d - 1) / 2 for any integer d.

Skew<T>(params IEnumerable<T>)

Generates a shear (skew) matrix using the provided angles.

public static Matrix<T> Skew<T>(params IEnumerable<T> angles) where T : struct, IFloatingPoint<T>, ITrigonometricFunctions<T>, IRootFunctions<T>

Parameters

angles IEnumerable<T>

One angle per off-diagonal coefficient of the base d × d shear block, in row-major order skipping the diagonal (row 0's d-1 off-diagonal columns, then row 1's, and so on). The base dimension d is inferred from the count: d * (d - 1) angles are required, since a full shear has one free coefficient per off-diagonal position.

Returns

Matrix<T>

A new (d+1) × (d+1) homogeneous shear matrix: identity everywhere except the supplied tan(angle) values at the base block's off-diagonal positions.

Type Parameters

T

Numeric type of the matrix.

Exceptions

ArgumentException

Thrown when the angle count does not match d * (d - 1) for any integer d.

Transform<T>(params IEnumerable<T>)

Generates a general affine transformation matrix.

public static Matrix<T> Transform<T>(params IEnumerable<T> values) where T : struct, IFloatingPoint<T>, IRootFunctions<T>

Parameters

values IEnumerable<T>

Coefficients for the upper d × (d+1) affine block, in row-major order: for each of the d base-dimension rows, d linear coefficients followed by that row's translation coefficient (last column). The base dimension d is inferred from the count: d * (d + 1) values are required. For example, a 2D transform needs 6 values [a, b, tx, c, d, ty], producing the matrix rows [a, b, tx], [c, d, ty].

Returns

Matrix<T>

A new (d+1) × (d+1) homogeneous transformation matrix: the supplied coefficients fill the upper affine block, and the final row is left as the homogeneous row [0, …, 0, 1], consistent with this library's matrix-times-column-vector multiplication convention (the same convention Translation<T>(params IEnumerable<T>) uses for its translation column).

Type Parameters

T

Numeric type of the matrix.

Exceptions

ArgumentException

Thrown when the value count does not match d * (d + 1) for any integer d.
