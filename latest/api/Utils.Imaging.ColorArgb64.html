Remarks

The overlapping Value field uses a [StructLayout(LayoutKind.Explicit)] layout that assumes little-endian byte order. On a little-endian host the packed value is ((ulong)Alpha << 48) | ((ulong)Red << 32) | ((ulong)Green << 16) | Blue. Since this library depends on System.Drawing (Windows/GDI+) it only runs on little-endian platforms, so the canonical packing and the overlapping layout are always consistent.
