Remarks

Wraps a List<T> rather than inheriting from it, so that the collection storage (data) stays separate from the bounding-box computation (processing logic exposed through IList<T>/IReadOnlyList<T>). All list operations still work exactly as before, including collection-initializer syntax (new GeoPointList<double> { p1, p2 }).
