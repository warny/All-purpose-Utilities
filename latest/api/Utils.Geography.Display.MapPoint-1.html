public MapPoint(ProjectedPoint<T> projectedPoint, byte zoomLevel, int tileSize)

Parameters

projectedPoint ProjectedPoint<T>

Projected map coordinates.

zoomLevel byte

Target zoom level.

tileSize int

Tile size in pixels.

Remarks

Uses Normalize(ProjectedPoint<T>) (via Projection) to convert the projected point into [0,1] map-fraction coordinates before scaling by the total map size in pixels (tileSize << zoomLevel, computed in long to avoid overflowing int at high zoom levels), matching GetMapSize(byte) and MappointToTile(ProjectedPoint<T>, byte) exactly, so both paths agree on which tile a given point falls into. The resulting pixel coordinates are clamped to [0, mapSize - 1]: a normalized value of exactly 1 (e.g. Equirectangular's latitude 90°, or Mercator's own MaxLatitude) would otherwise floor to mapSize, one pixel past the last valid one.
