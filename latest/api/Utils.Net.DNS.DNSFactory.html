Provides a mechanism to discover and store DNS-related types, along with their associated class and record IDs. This class uses DNSRecordAttribute metadata to map DNS record classes and IDs to corresponding .NET types that implement DNSResponseDetail.

Remarks

A default singleton instance is available via Default. Additional instances can be built from specified assemblies or types. Instances may be combined with the + operator to aggregate multiple DNSFactory configurations into one.

DNSFactory(params IEnumerable<DNSFactory>)

Initializes a new instance of the DNSFactory class by aggregating one or more DNSFactory instances into a single combined factory.

public DNSFactory(params IEnumerable<DNSFactory> factories)

Parameters

factories IEnumerable<DNSFactory>

A collection of DNSFactory instances to combine.

Remarks

The aggregated factory merges the mappings from all specified factories. If duplicates occur for the same (DNSClassId, ID) or (DNSClassId, name) pair, the last entry encountered overwrites previous ones.
