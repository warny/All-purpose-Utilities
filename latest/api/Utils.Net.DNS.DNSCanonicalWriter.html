Remarks

The DNSCanonicalWriter scans assemblies (via DNSFactory) for DNSResponseDetail types marked with DNSRecordAttribute. It then generates expression-based writers that serialize each DNS element's fields into the DNS wire format. When writing a DNS packet:

  • The DNS header is written first, updating record counts (QDCount, ANCount, etc.).
  • Each request record is written, translating its string Type to a numeric code (requestClassTypes).
  • Each response record is written, including the user-defined RData portion.

The final output is a byte[] representing the DNS packet to be sent over the network or stored.
