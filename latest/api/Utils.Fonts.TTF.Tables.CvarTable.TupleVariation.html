Properties

Deltas

Gets or sets the delta values (signed 16-bit) to add to each targeted CVT entry. When PointNumbers is null, the array is indexed directly by CVT index; otherwise Deltas[i] applies to PointNumbers[i].

public short[] Deltas { get; set; }

IntermedEndCoords

Gets the end coordinates of an intermediate region, or null if this is a non-intermediate region. Length must equal AxisCount when set.

public short[] IntermedEndCoords { get; init; }

short[]

IntermedStartCoords

Gets the start coordinates of an intermediate region, or null if this is a non-intermediate (peak-only) region. Length must equal AxisCount when set.

public short[] IntermedStartCoords { get; init; }

Property Value

short[]

PeakCoords

Gets the peak coordinates of the variation region, one F2Dot14 value per design axis. F2Dot14 is a signed 2.14 fixed-point number: divide by 16 384 to get the real value (e.g. 0x4000 = 1.0, 0x0000 = 0.0, 0xC000 = –1.0). Length must equal AxisCount.

public short[] PeakCoords { get; init; }

Property Value

short[]

PointNumbers

Gets or sets the CVT indices to which Deltas apply, or null to indicate that deltas apply to all CVT entries (in order).

public int[] PointNumbers { get; set; }
