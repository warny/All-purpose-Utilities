Decrement<T>(T?)

Decrements an endpoint if finite. If already null => -∞ or +∞? We have to interpret usage.

This method is used for subtracting an interval: we want (r2.Min - 1) etc. But if r2.Min = null => that's -∞ => (r2.Min - 1) => -∞ => no difference.
