Remarks

Derive from this class to define XML-processing logic. Annotate your handler methods with MatchAttribute to match specific XPath expressions, and optionally use XmlNamespaceAttribute at the class level to register namespace prefixes.

Remarks

This method disables DTD processing, clears the XML resolver, and limits entity expansion to reduce XML external entity and expansion attacks when processing untrusted inputs.
