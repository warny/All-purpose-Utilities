Class BoundedStack<T>

Namespace
Utils.VirtualMachine
Assembly
Utils.VirtualMachine.dll

A stack with a configurable maximum depth. Push throws InvalidOperationException when the depth limit is reached; Pop and Peek throw when the stack is empty. All exceptions carry VM-specific messages that include the depth limit for diagnostics.
