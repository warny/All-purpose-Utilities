MappointToTile(ProjectedPoint<T>, byte)

Converts a projected map point to tile coordinates at the specified zoom level.

public Tile<T> MappointToTile(ProjectedPoint<T> projectedPoint, byte zoomLevel)

Parameters

projectedPoint ProjectedPoint<T>

Projected map point.

zoomLevel byte

Zoom level of the tile grid.

Returns

Tile<T>

The tile containing the projected point.

Remarks

Uses Normalize(ProjectedPoint<T>) to convert the projected point into [0,1] map-fraction coordinates before scaling by GetMapSize(byte), matching MapPoint<T>'s own pixel calculation exactly, so both agree on which tile a given point falls into. The resulting pixel coordinates are clamped to [0, mapSize - 1] before dividing by TileSize: a normalized value of exactly 1 (e.g. Equirectangular's latitude 90°, or Mercator's own MaxLatitude) would otherwise floor to mapSize, one tile past the last valid one — the same edge case MapPoint<T> guards against.
