Remarks

The floating-point API mirrors the integer-based DrawI<T> helpers while transparently converting coordinates expressed in an arbitrary viewport to pixel positions. This allows callers to operate in normalized or physical spaces without manually performing the conversions required by the underlying IImageAccessor<T> implementation.

All conversions clamp to the image bounds when delegated to Draw, ensuring consistency with the low-level routines.
