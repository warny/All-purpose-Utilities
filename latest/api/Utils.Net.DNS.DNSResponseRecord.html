Remarks

A DNS response record includes:

  • The queried Name (DNSDomainName).
  • The numeric Class of the response (e.g., 1 for A, 28 for AAAA), stored in Utils.Net.DNS.DNSResponseDetail.ClassId.
  • The DNS class (ClassId), typically IN.
  • The TTL, or time-to-live, indicating how long the record may be cached.
  • An RDLength (2 bytes) indicating the size of the resource data.
  • The RData, which is a DNSResponseDetail object representing the record-specific data (e.g., IP addresses, mail server, etc.).

The wire layout in RFC 1035 §4.1.3 is:

+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+
/                      NAME                     /
+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+
|                      TYPE                     |
+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+
|                     CLASS                     |
+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+
|                      TTL                      |
+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+
|                   RDLENGTH                    |
+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--|
/                     RDATA                     /
+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+
