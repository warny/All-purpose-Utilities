Remarks

Computed via a scaled sum-of-squares accumulation (the same running-scale technique as BLAS nrm2/hypot), rather than summing component * component directly. A direct sum can overflow to infinity for large-but-representable components (whose square individually overflows even though the true norm does not), or underflow to zero for small-but-representable components (whose square underflows to zero even though it contributes to a representable norm). Tracking the largest component seen so far as a running scale and accumulating only the ratio of each component to that scale keeps every intermediate value within a representable range.

CrossProduct(params Vector<T>[])

Returns the cross product of (n-1) vectors of dimension n.

public static Vector<T> CrossProduct(params Vector<T>[] vectors)

Parameters

vectors Vector<T>[]

Vectors of dimension n.

Returns

Vector<T>

A vector perpendicular to all input vectors.

Remarks

Each component result[i] is (-1)^i times the determinant of the (n-1)x(n-1) minor formed by the input vectors' components, excluding column i (the standard Laplace expansion of the cross product along a virtual first row of basis vectors). Each minor's determinant is computed via Determinant, which uses Gaussian elimination (O(n^3)), rather than the previous direct recursive cofactor expansion (O(n!)) that also re-filtered the remaining-columns sequence at every recursion level.

Exceptions

ArgumentException

Thrown if vectors are not all of dimension n.

FromNormalSpace(T?)

Converts a vector usable in normal space to a vector usable in Cartesian space.

public Vector<T> FromNormalSpace(T? tolerance = null)

Parameters

tolerance T?

Overrides the default relative-plus-absolute tolerance (see Utils.Mathematics.LinearAlgebra.Vector<T>.DefaultNormTolerance(T[])) below which the homogeneous coordinate is treated as zero. Pass T.Zero to reject only an exactly-zero coordinate. Must be finite and non-negative when supplied.

Returns

Vector<T>

The converted vector for Cartesian space.

Exceptions

InvalidOperationException

Thrown when the homogeneous coordinate (last component) is zero or, absent an explicit tolerance, numerically negligible relative to the vector's components (e.g. w = 1e-300, which would otherwise silently divide the other components up to an astronomically large, meaningless magnitude instead of being recognized as a direction at infinity). A near-zero homogeneous coordinate represents a direction at infinity, not a point, and has no meaningful Cartesian equivalent.

ArgumentOutOfRangeException

Thrown when tolerance is supplied but not finite or is negative.
