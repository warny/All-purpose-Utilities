Class PcltTable

Namespace
Utils.Fonts.TTF.Tables
Assembly
Utils.Fonts.dll

The 'PCLT' table contains PCL 5 characterization data: information needed to identify and render the font on PCL 5-capable printers. The table has a fixed size of 54 bytes.

[TTFTable(TableTypes.Tags.PCLT, new TableTypes.Tags[] { })]
