Remarks

This struct splits the domain name into two parts: the immediate (leftmost) subdomain and the optional remaining parent domain. For example, sub.example.com is split into SubDomain = "sub" and ParentDomain = "example.com". It offers convenient methods and operators to facilitate domain concatenation, comparison, and equality checks.
