Class DNSDatagram

Namespace
Utils.Net.DNS
Assembly
Utils.Net.dll

Represents a DNS datagram, which provides methods for reading and writing DNS-related data. It supports simple DNS name compression, reading and writing integers of various sizes, and handling string labels in DNS-compatible formats.
