public ReplacementRule(string oldValue, string newValue, ReplacementScope scope, IntRange<long>? onScale, IntRange<long>? onValue)

Parameters

oldValue string

The text that should be replaced.

newValue string

The replacement text.

scope ReplacementScope

The scope that controls how the replacement is applied.

onScale IntRange<long>

Scale level(s) this rule is restricted to, or null for all levels. 0 = units, 1 = thousands, 2 = millions, etc. Supports range syntax ("1..3", "2,4..").

onValue IntRange<long>

Optional numeric-value filter. When set with onScale, the rule only fires when the group's numeric value falls within this range. When set without onScale, applies to the full number in the final assembled-string pass.

Exceptions

ArgumentException

Thrown when oldValue or newValue is null or empty.
