Remarks

The DrawI class exposes low-level rasterization primitives. It is intentionally conservative about bounds checking and delegates all color decisions to IBrush<T> implementations so that higher level helpers, such as DrawF<T>, can reuse the same pixel pipeline.

The accessor reference is stored for the whole lifetime of the helper so that callers can perform several drawing operations without recreating the wrapper.

DrawShapeThick(Func<float, float, T>, float, JoinStyle, IEnumerable<IDrawable>)

Draws one or more shapes with a given stroke width and join style, using a color function that receives the arc length from the start and the signed perpendicular distance from the centre line.

public void DrawShapeThick(Func<float, float, T> color, float width, JoinStyle join, IEnumerable<IDrawable> drawables)

Parameters

color Func<float, float, T>

Delegate (arcLength, perpDist) → T:

  • arcLength — accumulated path length from the start of the shape to the current pixel, in pixels.
  • perpDist — signed perpendicular distance from the centre line, in pixels. Positive values are on the left side of the stroke direction; negative on the right. Ranges in [−width/2, +width/2] along segments; uses outerSign × distanceFromJoint in join areas.
width float

Total stroke width in pixels.

join JoinStyle

Join style applied at every vertex where two segments meet.

drawables IEnumerable<IDrawable>

Shapes to stroke.
