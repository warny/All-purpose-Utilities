Remarks

Construct via FromSpan(Vector<T>, params Vector<T>[]) (k spanning vectors → dimension k) or FromNormals(Vector<T>, params Vector<T>[]) (k normal vectors → dimension d − k). The internal representation is an orthonormal basis computed by Gram–Schmidt, so all geometric operations are numerically stable.

Special cases: dimension 0 is a single point; dimension d is the full ambient space.

IntersectWith(AffineSubspace<T>)

Computes the intersection of this subspace with another affine subspace.

public AffineSubspace<T>? IntersectWith(AffineSubspace<T> other)

Parameters

other AffineSubspace<T>

The other affine subspace. Must share the same ambient dimension.

Returns

AffineSubspace<T>

The intersection as a new AffineSubspace<T>, or null when the two subspaces are parallel and disjoint.

Remarks

Combines the normal constraints of both subspaces using an augmented Gram–Schmidt that detects inconsistency (parallel subspaces with no common point). The anchor of the returned subspace is the minimum-norm solution of the combined constraint system.

Exceptions

ArgumentException

Thrown when other has a different ambient dimension.
