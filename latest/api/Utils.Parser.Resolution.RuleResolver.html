Class RuleResolver

Utils.Parser.Resolution

Utils.Parser.dll

Validates a ParserDefinition and enriches it with derived information.

Resolution is a two-step process:

  1. Build the flat AllRules lookup from all modes and parser-rule lists, assigning RuleKind values.
  2. Validate all RuleRef targets, perform consistency checks (mixed lexer/parser content), verify fragment constraints, and validate labels.
