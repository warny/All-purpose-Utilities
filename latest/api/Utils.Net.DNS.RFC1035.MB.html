Remarks

The MB record specifies a domain name (represented by MadName) that hosts a particular mailbox. A DNS resolver processing an MB record is advised to look up corresponding A or AAAA records for the host identified in MadName.

MB is part of an older or less commonly used set of mail-related DNS records (along with MG and MR). Modern mail routing primarily relies on MX records, but MB is included here for completeness in certain experimental or legacy scenarios.

Type code is set to 0x07 for class IN, as per RFC 1035 §3.3.3.

The experimental RDATA layout is:

+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+
/                   MADNAME                     /
/                                               /
+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+

MADNAME is the host that holds the specified mailbox; resolvers often fetch A/AAAA glue.
