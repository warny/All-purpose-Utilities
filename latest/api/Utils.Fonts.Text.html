Remarks

The class iterates over each character, retrieves its glyph via GetGlyph(char), applies kerning corrections via GetSpacingCorrection(char, char), and collects the resulting screen-space Path objects. All paths are then exposed through the IDrawable interface so that a single fill call covers the whole string.

UV coordinates produced by the fill are normalised over the bounding box of the entire text block, making texture or gradient effects straightforward:

draw.FillShape2((u, v) => texture[(int)(u * w), (int)(v * h)], text);

Parameters

content string

The string to render.

font IFont

Font used to retrieve glyphs and kerning data.

x float

Horizontal pen position of the first character in pixels.

baselineY float

Screen-space Y coordinate of the text baseline. Typically computed as topY + ascent * scale using the font's ascender value.

scale float

Factor mapping font units to pixels, usually desiredPixelHeight / unitsPerEm. The Y axis is automatically flipped to convert from TrueType space (Y-up) to screen space (Y-down).
