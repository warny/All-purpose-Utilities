SplitTopLevel(string)

Splits raw rule-call argument text into top-level argument slices at commas, respecting nested parentheses, square brackets, braces, and quoted strings.

public static IReadOnlyList<string> SplitTopLevel(string rawArguments)

Parameters

rawArguments string

Raw argument text without the outer [ and ] delimiters, as stored in RawArguments.

Returns

IReadOnlyList<string>

A read-only list of trimmed argument slices. Returns an empty list when rawArguments is empty or contains only whitespace. Empty segments (e.g. from a,,b or a trailing comma) are preserved.

Remarks

Nesting is tracked for (), [], and {}. Angle brackets <> are not tracked to avoid misinterpreting comparison operators.

Quoted strings ("..." and '...') are respected; backslash escapes (\", \', \\) inside strings are recognized. Verbatim C# strings (@"...") are not supported.

Unbalanced brackets or quotes are handled conservatively: the remainder of the input is absorbed into the current segment without throwing.

This method does not evaluate expressions, infer types, or bind arguments to parameters.

Exceptions

ArgumentNullException

Thrown when rawArguments is null.
