Namespace Utils.Net.DNS.RFC2535

Classes

KEY

Represents a DNS KEY record (type = 25) as specified in RFC 2535 §3. The KEY record stores a public key and associated metadata (flags, protocol, algorithm).

NXT

Represents an NXT (Next) record in DNS as specified by RFC 2535 §5.1.

SIG

Represents a SIG (Signature) record in DNS as defined in RFC 2535. The SIG record is the fundamental mechanism by which data is authenticated in the secure DNS. It digitally signs an RRset (as defined in RFC 2181) and binds the data to a specific time interval and the signer's domain name.
