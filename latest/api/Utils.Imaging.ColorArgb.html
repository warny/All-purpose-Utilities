Gradient(ColorArgb, ColorArgb, double)

Blends two colors using a perceptual (square-root) gradient.

public static ColorArgb Gradient(ColorArgb color1, ColorArgb color2, double percent)

Parameters

color1 ColorArgb

The starting color.

color2 ColorArgb

The ending color.

percent double

Blend factor clamped to the [0,1] range.

Returns

ColorArgb

The interpolated color.

Remarks

RGB channels are interpolated in the power-2 domain and projected back via square root, which produces a more visually uniform gradient than linear interpolation. Alpha is interpolated linearly. For a linear interpolation, use LinearGradient(ColorArgb, ColorArgb, double).
