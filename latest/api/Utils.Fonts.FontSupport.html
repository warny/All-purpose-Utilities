Class FontSupport

Namespace

Assembly

Provides utilities and encoding tables for working with font character sets and glyph name mappings. Includes standard encodings (WinAnsi, MacRoman, ISO Latin1, etc.) and predefined glyph names used in Type 1 and Type 1C fonts.
