Remarks

All structural capacities must be at least one. Hard implementation ceilings (e.g. MaxValue identifier exhaustion) remain separate from caller-configured limits stored here. Use Default for behavior identical to unconfigured classes.

Constructors

Initializes a new VirtualMachineLimits with the specified capacities. All parameters default to values matching the current unconfigured behavior.

public VirtualMachineLimits(int maxCallStackDepth = 512, int maxControlFlowDepth = 1024, int maxOperandStackDepth = 1024, int maxPhysicalPages = 2147483647, int maxMemoryProcesses = 2147483647, int maxScheduledProcesses = 2147483647, int schedulerQuantumSteps = 100)

Parameters

maxCallStackDepth int

Maximum call-stack depth. Must be at least 1.

maxControlFlowDepth int

Maximum control-flow nesting depth. Must be at least 1.

maxOperandStackDepth int

Maximum operand-stack depth. Must be at least 1.

maxPhysicalPages int

Maximum simultaneously allocated pages. Must be at least 1.

maxMemoryProcesses int

Maximum simultaneously registered memory processes (master included). Must be at least 1.

maxScheduledProcesses int

Maximum simultaneously registered scheduled processes. Must be at least 1.

schedulerQuantumSteps int

Maximum instructions per process per scheduler step. Must be at least 1.

Exceptions

ArgumentOutOfRangeException

Thrown when any parameter is less than one.
