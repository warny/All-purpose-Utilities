Remarks

Format 0 (simple): all glyphs share the same default property value — no per-glyph override.

Format 1 (lookup): an AAT lookup table maps individual glyphs to their property values. Glyphs not in the lookup use DefaultProp.

Property bit layout (UInt16):

  • Bits 15–14: direction (0=strong L→R, 1=strong R→L, 2=weak, 3=neutral).
  • Bit 13: hangs on right.
  • Bit 12: hangs on left.
  • Bit 11: suppress kerning before.
  • Bit 10: suppress kerning after.
