The 'OS/2' table contains metrics and other data required for correct rendering on Windows and OS/2 platforms. It is present in virtually every TrueType/OpenType font and describes properties such as weight, width, style, typographic metrics, Unicode character ranges and vendor identification.

Remarks

The table exists in six versions (0–5) that grow in size; ReadData(Reader) and WriteData(Writer) adapt automatically to the stored Version.

  • v0 — 78 bytes (base)
  • v1 — 86 bytes (adds code-page ranges)
  • v2/v3/v4 — 96 bytes (adds typographic height, cap height, break/default chars)
  • v5 — 100 bytes (adds optical size range)
