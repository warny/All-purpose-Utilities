Returns the spacing correction (kerning) between two adjacent characters. If a kern table is present, its value is returned; otherwise, 0 is returned.

Returns

The spacing correction in font units.

Remarks

The 'kern' table stores kerning pairs by glyph index, never by character code, so before and after are resolved through 'cmap' first (the same resolution GetGlyph(char) performs) before consulting the kern table.
