Class HeadTable

Namespace
Utils.Fonts.TTF
Assembly
Utils.Fonts.dll

The 'head' table contains global information about the font. It records facts such as the font version number, creation and modification dates, revision number, and basic typographic data that applies to the font as a whole. This includes the font bounding box, the primary writing direction, and other information regarding glyph placement within the em square. The checksum is used to verify the integrity of the font data and can help distinguish between similar fonts.
