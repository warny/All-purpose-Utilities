Convert(decimal, int, params string[])

Converts a decimal value into its string representation with a mandatory number of decimal digits, applying optional variant parameters.

When mandatoryDecimalDigits is negative, the decimal part is shown as-is (same as Convert(decimal, params string[])). When zero, the decimal part is suppressed (only the integer part is shown). When positive, the value is rounded to that many decimal places and the decimal part is always shown with exactly that many digits, padding with zeros if needed.

The default implementation ignores precision and variant parameters and delegates to Convert(decimal); implementations should override.

string Convert(decimal number, int mandatoryDecimalDigits, params string[] variants)

Parameters

number decimal

The value to convert.

mandatoryDecimalDigits int

Number of decimal digits to always show. Negative = natural (as-is), 0 = integer only, positive = always show exactly N decimal digits (rounded and zero-padded).

variants string[]

Zero or more "dimension=value" strings.

Returns

string

The formatted number with the requested precision and variants applied.

Converts a decimal value into its string representation with a mandatory number of decimal digits, custom decimal formatting options, and optional variant parameters. The default implementation ignores all parameters and delegates to Convert(decimal); implementations should override.

Negative: show the decimal part as-is. Zero: suppress the decimal part entirely. Positive: round to N decimal places and always show exactly N digits (zero-padded).

Optional overrides for the decimal separator word (DecimalSeparator), the denomination suffix (DecimalSuffix), and zero-decimal suppression (OmitZeroDecimals). Both "(s)" markers are pluralized: the separator against the integer part, the suffix against the decimal value.
