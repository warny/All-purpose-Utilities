Namespace Utils.Net.DNS

Classes

DNSCanonicalWriter

Provides functionality to serialize DNS packets into a byte array using reflection and expression trees. This class can handle standard DNS headers, request records, response records, and user-defined DNS response details annotated with DNSRecordAttribute.

DNSConstants

Constants representing DNS datagram flags and fields masks.

DNSDatagram

Represents a DNS datagram, which provides methods for reading and writing DNS-related data. It supports simple DNS name compression, reading and writing integers of various sizes, and handling string labels in DNS-compatible formats.

DNSElement

Represents the abstract base class for all DNS elements within this library.

DNSElementsComparer

Provides an IEqualityComparer<T> implementation for DNSElement objects.

DNSFactory

Provides a mechanism to discover and store DNS-related types, along with their associated class and record IDs. This class uses DNSRecordAttribute metadata to map DNS record classes and IDs to corresponding .NET types that implement DNSResponseDetail.

DNSFieldAttribute

Applied to properties or fields within a DNS record class to indicate how the corresponding data should be serialized, deserialized, or otherwise interpreted.

DNSHeader

Represents the DNS header section of a DNS packet, including standard bits from RFC 1035 §4.1.1 and the AD/CD extensions introduced in RFC 2535. The header provides information about flags, counts for queries/answers/authorities/additionals, and other DNS-level metadata.

DNSHeadersComparer

Provides an IEqualityComparer<T> implementation for DNSHeader objects.

DNSPacketReader

Provides helpers to translate DNS datagrams into strongly typed response structures.

DNSPacketWriter

Provides functionality to serialize DNS packets into a byte array using reflection and expression trees. This class can handle standard DNS headers, request records, response records, and user-defined DNS response details annotated with DNSRecordAttribute.

DNSRecordAttribute

Specifies a DNS record attribute that can be applied at the class level to identify the DNS DNSClassId, record ID, and an optional descriptive name.

DNSRequestRecord

Represents a DNS query (request) record within a DNS packet, following the layout defined in RFC 1035 §4.1.2 and indicating which record type (e.g., A, AAAA, MX) is being queried for a particular domain name.

DNSRequestType

Request type identifiers used in legacy AXFR-style zone transfers.

DNSResponseDetail

Represents the base class for DNS response details (RData). Concrete types that inherit from this class are annotated with DNSRecordAttribute for identifying DNS record types and include fields marked by DNSFieldAttribute for serialization.

DNSResponseRecord

Represents a DNS response record, containing all necessary fields to describe a specific DNS answer in a response packet, including the domain name, the record type, TTL, and the RData, as defined by RFC 1035 §4.1.3.

DNSText

Provides helper methods to convert DNS records to and from the textual representation commonly used in zone files.

DNSTextRecordAttribute

Specifies the textual format for a DNS record. The format string may contain property or field names enclosed in curly braces (e.g. {Preference}) that will be replaced when converting to or from text.

Structs

DNSDomainName

Represents a DNS domain name composed of an optional subdomain and a parent domain. Provides comparison and equality checks against both string and DNSDomainName.

Describes a component capable of reading DNS headers from a transport-specific source.
