Remarks

Implement this interface alongside INumberToStringLanguageSpecifics on the same class. When the configured LanguageSpecifics object implements this interface, ConvertOrdinal(int, params string[]) invokes TryConvertOrdinal(int, IReadOnlyDictionary<string, string>, out string?) before consulting the XML pipeline. Returning true short-circuits exceptions, word rules, and suffix.

Example use cases: Polish (case/gender/number agreement), Russian (same), Arabic (root-pattern transformations), German (full adjectival declension).

Parameters

The absolute value of the number to convert (always ≥ 0).

activeVariants IReadOnlyDictionary<string, string>

The active dimension constraints, e.g. { "gender" → "feminin", "case" → "dativ" }. Populated by merging the caller's explicit parameters with dimension defaults.

result string

The ordinal string when this method returns true; otherwise null.

Returns

bool

true to use result directly; false to fall through to the XML-based ordinal pipeline.

TryConvertOrdinal(long, IReadOnlyDictionary<string, string>, out string?)

Attempts to convert a 64-bit integer to its ordinal string representation. The default implementation delegates to TryConvertOrdinal(int, IReadOnlyDictionary<string, string>, out string?) when the value fits in int; values above MaxValue return false (fall through to the XML ordinal pipeline). Override this to handle ordinal values above MaxValue.
