Represents an RP (Responsible Person) record, as defined in RFC 1183 §2.2. This record identifies the mailbox of the person responsible for a particular DNS zone and an optional TXT record with additional information.

Remarks

The RP record contains two domain names:

  • MBoxDName: A domain name that specifies the mailbox for the responsible person. By convention, the first label of the name is used to replace the '.' with '@' to form a typical email address. For example, john.doe.example.com might map to john.doe@example.com.
  • TxtDName: A domain name for a TXT record containing more information (e.g., about the responsible person). If this name points to the root (e.g., .), it indicates that no additional information is available.

An example usage might be: 

example.com.  86400  IN  RP  john.doe.example.com.  info.example.com.

This indicates that john.doe@example.com is responsible for example.com, and info.example.com might hold a TXT record with further contact or host data.

The wire format contains the two domain names consecutively:

+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+
/                 MBOXDNAME                    /
+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+
/                 TXTSDNAME                    /
+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+

Gets or sets the domain name that points to a TXT record holding more information about the responsible person or host. If this is set to the root label ('.'), it indicates there is no additional information.
