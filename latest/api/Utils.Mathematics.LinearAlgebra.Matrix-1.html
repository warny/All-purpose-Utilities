ComputeEigenvalues(int)

Computes the real eigenvalues and corresponding eigenvectors of a symmetric matrix using the QR iteration algorithm.

public (T[] Eigenvalues, Matrix<T> Eigenvectors) ComputeEigenvalues(int maxIterations = 1000)

Parameters

maxIterations int

Maximum number of QR iterations before giving up.

Returns

(T[] Eigenvalues, Matrix<T> Eigenvectors)

A tuple containing an array of eigenvalues (descending by magnitude) and a matrix whose columns are the corresponding eigenvectors.

Remarks

Only real symmetric matrices are supported; all eigenvalues of such matrices are guaranteed real. Eigenvalues are returned in descending order of absolute value.

Exceptions

InvalidOperationException

Thrown when the matrix is not square, not symmetric, or fails to converge.

DecomposeQR()

Performs a QR decomposition of this matrix using the modified Gram–Schmidt algorithm.

public (Matrix<T> Q, Matrix<T> R) DecomposeQR()

Returns

(Matrix<T> L, Matrix<T> U)

A tuple containing the orthogonal matrix Q and the upper-triangular matrix R.

Remarks

For an m×n matrix A with m ≥ n and full column rank, returns Q (m×n, orthonormal columns) and R (n×n, upper triangular) such that A = Q·R.

Exceptions

InvalidOperationException

Thrown when the matrix has more columns than rows, or when its columns are not linearly independent.

DiagonalizeLU()

Performs LU decomposition of the current square matrix, resulting in a lower triangular matrix L and an upper triangular matrix U such that the original matrix equals L multiplied by U.

public (Matrix<T> L, Matrix<T> U) DiagonalizeLU()

Returns

(Matrix<T> L, Matrix<T> U)

A tuple containing the lower and upper triangular matrices.

Remarks

The decomposition uses the same sequence of elementary row operations as the original mutable implementation but operates entirely on local array copies to preserve immutability.

Exceptions
