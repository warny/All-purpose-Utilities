ParseRangeExpression(string)

Parses a comma-separated range expression in the XML attribute syntax and returns an IntRange<long> that can be used with Contains(T).

public static IntRange<long> ParseRangeExpression(string s)

Parameters

s string

Returns

IntRange<long>

Remarks

Accepted segment syntax (comma-separated list):

  • 5 — exact value 5
  • 1..3 — inclusive range [1, 3]
  • ..5 — all values ≤ 5 (open lower bound)
  • 5.. — all values ≥ 5 (open upper bound)

Example: "1,5..10,20.." matches 1, five through ten, and any value ≥ 20.

Exceptions

ArgumentException

Thrown when s is null or whitespace.

FormatException

Thrown when a segment cannot be parsed.
