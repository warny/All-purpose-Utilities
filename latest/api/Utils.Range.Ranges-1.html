Class Ranges<T>

Namespace
Utils.Range
Assembly
Utils.dll

Represents a set of intervals (ranges) for an ordered type T. Provides set-like operations (union, intersection, difference, symmetric difference) via bitwise operators: | => Union, & => Intersection, ^ => Symmetric Difference, ~ => Complement (relative to entire domain, if desired),

  • => Difference (Except).

The intervals are stored internally as disjoint, sorted Range<T> objects.

public class Ranges<T> : IFormattable, IEquatable<Ranges<T>>, ISubtractionOperators<Ranges<T>, Ranges<T>, Ranges<T>>, IBitwiseOperators<Ranges<T>, Ranges<T>, Ranges<T>>, IEqualityOperators<Ranges<T>, Ranges<T>, bool> where T : IComparable<T>
